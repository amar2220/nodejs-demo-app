# NodeJS Demo App - CI/CD Pipeline

## Objective
Automate code deployment using GitHub Actions.

## Tools Used
- GitHub
- GitHub Actions
- Node.js
- Docker

## Steps Performed
1. Created a Node.js application.
2. Added a Dockerfile.
3. Created GitHub Actions workflow file (main.yml).
4. Automated build process on push to main branch.

## Result
CI/CD pipeline executed successfully with green status.
